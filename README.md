# LaunchPixel

LaunchPixel is a cutting-edge agency site specializing in AI Automation, Web Development, and Digital Strategy.

## Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion, Three.js (React Three Fiber)
- **deployment**: Vercel / Static Export ready

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Features
- **3D Backgrounds**: Custom interactive particle systems (Antigravity).
- **Glassmorphism UI**: Premium dark-mode aesthetic with Spotlight effects.
- **SEO Optimized**: Fully configured Sitemap, Robots.txt, JSON-LD, and Manifest.
- **LLM Ready**: Includes `llms.txt` for better AI discovery.
